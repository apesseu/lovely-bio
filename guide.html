Guide d'utilisation complet

Bienvenue dans le guide d'utilisation de votre site web Lovely Bio. Ce document vous aidera à naviguer, personnaliser et tirer le meilleur parti de votre site vitrine.

1. Navigation sur le site

Votre site Lovely Bio est conçu pour être intuitif et facile à naviguer, tant pour vous que pour vos visiteurs.

Structure du site

Le site est organisé en plusieurs sections accessibles depuis le menu de navigation :

  • Accueil - La page d'accueil présente une introduction à votre marque et met en avant vos principaux produits.
  • À propos - Cette section raconte votre histoire, de Maroua/Moudan à aujourd'hui.
  • Produits - Présentation de vos produits cosmétiques naturels classés par catégories.
  • Services - Informations sur vos services de coaching et formation.
  • FAQ - Réponses aux questions fréquemment posées par vos clients.
  • Contact - Formulaire de contact et liens vers vos réseaux sociaux.

Navigation mobile

Sur les appareils mobiles, le menu est accessible en cliquant sur l'icône de menu "hamburger" (☰) en haut à droite de l'écran.

Astuce : Pour un défilement plus rapide, utilisez les liens de navigation plutôt que de faire défiler manuellement la page.

2. Personnalisation du site

Votre site est entièrement personnalisable pour refléter l'identité visuelle de votre marque Lovely Bio.

Modification des images

Pour remplacer les images par défaut par vos propres photos :

  1. Préparez des images de haute qualité dans les formats .jpg, .png ou .webp
  2. Créez un dossier "images" à la racine du site s'il n'existe pas déjà
  3. Nommez vos images avec des noms pertinents (par exemple : "hero-image.jpg", "savon-1.jpg", etc.)
  4. Replacez les images existantes dans le dossier "images" par vos propres fichiers

Modification des couleurs

Pour modifier les couleurs du site :

  1. Ouvrez le fichier "css/styles.css"
  2. Localisez la section ":root" au début du fichier
  3. Modifiez les valeurs des variables de couleurs selon vos préférences :
:root {
    --primary-color: #e68a00;    /* Couleur principale */
    --secondary-color: #8c5e11;  /* Couleur secondaire */
    --accent-color: #f8c471;     /* Couleur d'accent */
    /* ... autres variables ... */
}

Modification du contenu

Pour modifier les textes et informations du site :

  1. Ouvrez le fichier "index.html" avec un éditeur de texte
  2. Localisez la section à modifier (Accueil, À propos, Produits, etc.)
  3. Modifiez le texte entre les balises HTML correspondantes

3. Fonctionnalités de partage

Votre site inclut des fonctionnalités permettant aux visiteurs de le partager facilement sur différentes plateformes.

Options de partage

Dans la section Contact, les visiteurs peuvent partager votre site via :

  • Copier le lien - Copie l'URL du site dans le presse-papier
  • WhatsApp - Partage le lien via WhatsApp
  • Facebook - Partage le lien sur Facebook
  • Twitter - Partage le lien sur Twitter

Comment utiliser les boutons de partage

  1. Faites défiler jusqu'à la section Contact
  2. Cliquez sur l'un des boutons de partage dans la section "Partagez notre site"
  3. Suivez les instructions de la plateforme choisie pour finaliser le partage

Remarque : Les URL partagées incluront automatiquement le titre de votre site et son lien, optimisant ainsi la visibilité sur les réseaux sociaux.

4. Gestion des sections de produits

La section Produits de votre site permet aux visiteurs de filtrer vos offres par catégories.

Système de catégories

Les produits sont organisés par catégories :

  • Savons
  • Laits
  • Gels
  • Gommages

Ajout d'un nouveau produit

Pour ajouter un nouveau produit :

  1. Ouvrez le fichier "index.html"
  2. Localisez la section des produits (<section id="produits">)
  3. Repérez le conteneur des produits (<div class="products-grid">)
  4. Copiez un bloc de produit existant et collez-le dans le conteneur
  5. Modifiez les informations (image, titre, description, caractéristiques)
  6. Assurez-vous de définir correctement l'attribut data-category pour permettre le filtrage

Structure d'un bloc produit

<div class="product-card" data-category="savons">
    <div class="product-image">
        <img src="images/savon-1.jpg" alt="Savon naturel">
        <div class="product-overlay">
            <a href="#" class="view-product">Voir détails</a>
        </div>
    </div>
    <div class="product-info">
        <h3>Savon au Karité</h3>
        <p>Savon naturel enrichi au beurre de karité pour une peau douce et hydratée.</p>
        <div class="product-features">
            <span>100% Naturel</span>
            <span>Peau Sèche</span>
        </div>
    </div>
</div>

5. Gestion des formulaires

Votre site contient deux formulaires : un formulaire de contact et un formulaire d'inscription à la newsletter.

Formulaire de contact

Le formulaire de contact permet aux visiteurs de vous envoyer directement des messages. Pour le moment, il affiche simplement un message de confirmation après soumission.

Pour connecter ce formulaire à un véritable système d'e-mail :

  1. Ouvrez le fichier "js/main.js"
  2. Localisez la section de gestion du formulaire de contact (recherchez contactForm.addEventListener)
  3. Remplacez le code actuel par une requête AJAX vers votre service d'e-mail préféré ou un script PHP personnalisé

Formulaire Newsletter

De même, pour connecter le formulaire d'inscription à la newsletter à un service réel :

  1. Ouvrez le fichier "js/main.js"
  2. Localisez la section de gestion du formulaire newsletter (recherchez newsletterForm.addEventListener)
  3. Remplacez le code actuel par une intégration avec un service comme Mailchimp, SendinBlue, etc.

Important : Assurez-vous de respecter le RGPD en incluant des mentions légales appropriées concernant la collecte des données personnelles.

6. FAQ et réponses

La section FAQ permet de répondre aux questions fréquentes de vos clients et prospects.

Modification des questions existantes

Pour modifier les questions et réponses existantes :

  1. Ouvrez le fichier "index.html"
  2. Localisez la section FAQ (<section id="faq">)
  3. Modifiez le texte des questions (<h3>) et des réponses (<p>)

Ajout de nouvelles questions

Pour ajouter une nouvelle question :

  1. Copiez un bloc de question-réponse existant
  2. Collez-le dans le conteneur FAQ (<div class="faq-container">)
  3. Modifiez la question et la réponse selon vos besoins

7. Optimisation pour les moteurs de recherche (SEO)

Pour améliorer le référencement de votre site :

Métadonnées

Modifiez les balises meta dans la section <head> du fichier "index.html" :

  • Titre - Modifiez le contenu de la balise <title>
  • Description - Optimisez la balise <meta name="description">

Texte alternatif des images

Assurez-vous que toutes vos images ont un attribut alt descriptif et pertinent, par exemple :

<img src="images/savon-1.jpg" alt="Savon naturel au karité pour peau africaine">

Structure des en-têtes

Maintenez une hiérarchie logique des titres (h1, h2, h3) pour améliorer la compréhension de votre contenu par les moteurs de recherche.

8. Maintenance du site

Pour maintenir votre site à jour et performant :

Mises à jour régulières

  • Mettez à jour les informations de produits et services régulièrement
  • Actualisez les témoignages clients pour maintenir la confiance
  • Vérifiez et mettez à jour les liens vers vos réseaux sociaux

Vérification des performances

Utilisez des outils comme Google PageSpeed Insights ou GTmetrix pour évaluer et améliorer les performances de votre site.

Sauvegarde

Effectuez régulièrement des sauvegardes complètes de votre site (fichiers HTML, CSS, JavaScript et images).
